Secure Static Site

A production-grade static website hosted on AWS — deployed via CloudFormation and a GitHub Actions CI/CD pipeline.

S3 CloudFront WAFv2 ACM Route53 CloudFormation GitHub Actions OIDC Athena

Security Features

  • No public S3 access — CloudFront OAC only
  • TLS 1.2+ enforced, HTTP/3 enabled
  • WAFv2 with AWS managed rule groups
  • Security headers injected at the edge (CloudFront Function)
  • HSTS with preload, strict CSP, Permissions-Policy
  • OIDC authentication — no static AWS credentials
